# Personality Variant System

## How Variations Work

The canonical novella shows ONE playthrough. The game generates personality dynamically from these inputs, ranked by influence:

### Influence Hierarchy
1. **Player 1's base job** (strongest) — sets the core voice
2. **Playstyle** (very strong) — how you play shapes who you become
3. **Job evolution path** — twists the base in specific directions
4. **Secondary job** (moderate) — adds color, blends personality
5. **Tertiary job** (minimal) — mostly mechanical, slight flavor
6. **Other party members' configurations** — creates inter-party dynamics

### Base Job Personalities (Player 1)

| Job | Core Voice | Default Outlook | How They See the World |
|-----|-----------|----------------|----------------------|
| Fighter | Direct, earnest, action-first | Protective, duty-driven | Problems are things to stand against |
| Cleric | Attentive, patient, faith-driven | Caring, principled | Problems are people who need help |
| Mage | Analytical, curious, system-minded | Intellectual, pattern-seeking | Problems are puzzles to understand |
| Rogue | Perceptive, pragmatic, edge-finding | Opportunistic, survival-focused | Problems are systems with exploitable gaps |
| Bard | Expressive, observant, narrative-minded | Empathetic, story-driven | Problems are stories that need telling |

### Playstyle Modifiers

These compound with base job. A manual Fighter is different from an autobattle Fighter.

| Playstyle | Personality Drift | Key Dialogue Shifts |
|-----------|------------------|-------------------|
| Traditional manual | Present, engaged, values the moment | "I want to do this myself." |
| Heavy autobattle (self) | Detached from own combat, strategic | "The script handles it." |
| Full party autobattle | Architect mentality, watches from above | "We designed this. It runs." |
| Autogrind | Zen patience, comfortable with repetition | "Again. It's fine. Again." |
| One-shot builds | Perfectionist, impatient with inefficiency | "That should have been one hit." |
| Many-turn tactician | Patient, methodical, enjoys complexity | "Let's think about this." |
| Dialogue skipper | Terse, action-hungry, misses meta clues | "Can we go?" / discovers things late |
| Completionist | Thorough, can't leave things undone | "We should check that corner." |
| Side-quest focused | Wandering, curious, easily distracted | "What's over there?" |
| Speed runner | Impatient, efficient, cuts corners | Arrives at revelations faster but thinner |
| Conservative/defensive | Risk-averse, over-prepares | "Are we sure about this?" |
| Aggressive/risky | Bold, under-prepares, improvises | "We'll figure it out on the way." |
| Item hoarder | "Too good to use" syndrome | Never uses best items, wins with basics |

### Evolution Personality Twists

Job evolutions per world shift the base personality:

**Fighter evolutions:**
- Fighter → Knight-Errant: honorable, slightly naive, believes in the system
- Knight-Errant → Security Guard: procedural, follows rules he didn't write
- Security Guard → Steamknight: mechanical precision, starts questioning the machine
- Steamknight → Enforcer: institutional weight, realizes enforcement ≠ justice
- Enforcer → Firewall: rule-based thinking, rules become identity
- Firewall → Resolve: pure will, stripped to essence

**Cleric evolutions:**
- Cleric → Friar: humble, faith in people over institutions
- Friar → School Nurse: practical care, paperwork over prayer
- School Nurse → Steampriest: ceremony returns but feels hollow
- Steampriest → Safety Officer: care becomes compliance, fights this
- Safety Officer → Antivirus: healing as quarantine, clinical compassion
- Antivirus → Faith: pure conviction, the willingness to act without proof

**Mage evolutions:**
- Mage → Hedge Wizard: self-taught, scrappy, improvisational
- Hedge Wizard → Science Nerd: systematic, excited by mundane physics
- Science Nerd → Alchemist: free-form curiosity, mixing to discover
- Alchemist → Lab Technician: curiosity constrained by protocol, rebels
- Lab Technician → Compiler: writes reality, spells become code
- Compiler → Logic: pure reasoning, rigor as a form of love

**Rogue evolutions:**
- Rogue → Footpad: petty thief, stumbled into this, street-smart
- Footpad → Skater Kid: rebellious, anti-authority, fast
- Skater Kid → Saboteur: intentional disruption, finds purpose in breaking
- Saboteur → Whistleblower: breaking systems for justice, not fun
- Whistleblower → Exploit: finds gaps in everything, instinctive
- Exploit → Edge: pure boundary-awareness, the shape of limits

**Bard evolutions:**
- Bard → Minstrel: chronicler, narrates everything, earnest performer
- Minstrel → Band Kid: awkward, practicing, finding their voice
- Band Kid → Busker: independent, plays for everyone, answers to no one
- Busker → PA Announcer: voice co-opted by system, reclaims it
- PA Announcer → Broadcaster: signal vs noise, strategic expression
- Broadcaster → Voice: pure expression, naming what cannot be named

### Secondary Job Blends

The secondary job creates personality tension/harmony. Key combos:

| Primary + Secondary | Resulting Personality |
|-------------------|---------------------|
| Fighter + Cleric | Stoic protector, duty meets faith |
| Fighter + Mage | Tactical warrior, action meets analysis |
| Fighter + Rogue | Pragmatic survivor, honor meets opportunism |
| Fighter + Bard | Inspiring leader, action meets narrative |
| Cleric + Fighter | Militant healer, faith meets force |
| Cleric + Mage | Scholar-priest, faith meets intellect |
| Cleric + Rogue | Conflicted moralist, faith meets pragmatism |
| Mage + Fighter | Battle-scholar, analysis meets directness |
| Mage + Rogue | System hacker, understanding meets exploitation |
| Mage + Bard | Theatrical intellectual, analysis meets performance |
| Rogue + Fighter | Enforcer-type, edges meets duty |
| Rogue + Mage | Technical exploiter, gaps meets understanding |
| Rogue + Bard | Trickster storyteller, edges meets narrative |
| Bard + Fighter | War poet, expression meets action |
| Bard + Cleric | Hymn singer, expression meets devotion |
| Bard + Rogue | Con artist, expression meets deception |

### Key Divergence Scenes

These are the moments where personality variation has the most impact:

1. **Harmonia departure** — why does Player 1 answer the call?
2. **First Masterite defeat** — how do they react to the Warden?
3. **Mordaine confrontation** — what do they say to her?
4. **World transition shock** — how do they handle the genre shift?
5. **The Coordinator's monitor** — what do they notice first?
6. **The Calibrant reveal** — how quickly do they put it together?
7. **The Director's "I don't know"** — how do they respond?
8. **The Calibrant's question** — "What is a game without challenge?"
9. **The ending** — which of the four (or hybrid) do they get?
10. **Rest-point banter** — inter-party dynamics at inns/camps

### Alternate Playthrough Profiles

These represent distinct enough experiences to warrant their own novellas:

**"The Automator"** — Mage lead, heavy autobattle, analytical
- Sees the game as a system to understand and optimize
- Detached from combat, engaged with mechanics
- Figures out the Calibrant's pattern earliest (World 2)
- Gets the automation ending
- The Calibrant finds them the most threatening

**"The Faithful"** — Cleric lead, manual play, grind-heavy, completionist
- Sees the game as a world full of people to help
- Does every side quest, talks to every NPC
- Slowest to figure out the meta (distracted by caring)
- Gets the grind ending
- The Calibrant finds them the most confusing

**"The Breaker"** — Rogue lead, exploit-heavy, dialogue-skipper, speedrunner
- Sees the game as a system to crack open
- Breaks things the Calibrant didn't know could break
- Arrives at revelations through exploitation rather than dialogue
- Gets the exploit ending
- The Calibrant finds them the most frustrating

**"The Witness"** — Bard lead, traditional manual, completionist, never autobattles
- Sees the game as a story worth experiencing fully
- Narrates and documents everything
- Most emotionally affected by Masterite awakenings
- Gets the manual ending
- The Calibrant finds them the most unsettling (because they're paying attention)
